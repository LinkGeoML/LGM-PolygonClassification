polygon_feature_extraction.standardize_data_test(X, scaler)[source]¶

This function is responsible for standarizing a set of test data. It achieves this by using a MinMaxScaler object that was used previously for standardizing relevant train data. Ultimately, all features are within [0.0, 1.0]

Parameters

  • scaler (sklearn.preprocessing.MinMaxScaler object) – the MinMaxScaler object to be used for standardization

Returns

X – the standardized test data

numpy array
